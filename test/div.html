Lume UI Documentation

A comprehensive, modern web component library built with vanilla JavaScript, Shadow DOM, and designed for maximum flexibility and performance.

✨ Features

  • 🚀 Native Web Components - Built with modern Web Standards and Shadow DOM encapsulation for perfect isolation and performance
  • 🎨 Highly Customizable - Over 100+ styling properties and attributes per component with CSS-in-JS support
  • 📱 Responsive Design - Mobile-first approach with built-in breakpoints and adaptive layouts
  • 🌙 Dark/Light Themes - Built-in theme support with seamless switching and custom theme creation
  • ⚡ Performance Focused - Lightweight (~15KB gzipped) with lazy loading and tree-shaking support
  • 🔧 Framework Agnostic - Works seamlessly with React, Vue, Angular, Svelte, or vanilla JavaScript
  • 🎭 Multiple Variants - 7+ unique design variants per component with glassmorphism, gradients, and modern effects
  • 🎯 TypeScript Ready - Full type definitions, IntelliSense support, and comprehensive API documentation
  • 🧩 Modular Architecture - Import only what you need with individual component imports
  • ♿ Accessibility First - WCAG 2.1 compliant with full keyboard navigation and screen reader support

Installation

Multiple ways to get started with Lume UI - choose the method that best fits your project setup.

🚀 CDN (Recommended)

The fastest way to get started. Simply include the script tag and start using components immediately:

HTML 
<!-- Include Lume UI via CDN -->
<script type="module" src="https://cdn.jsdelivr.net/gh/Amrit-dev08/LumeUi/dist/lume.js"></script>

<!-- Start using components immediately -->
<lume-button variant="primary">Click me!</lume-button>
<lume-navbar logo-text="My App"></lume-navbar>

✨ CDN Benefits:

  • 🚀 Zero Setup - No build process required
  • 🌐 Global CDN - Fast delivery worldwide via jsDelivr
  • 📦 Auto Updates - Always get the latest stable version
  • ⚡ Optimized - Pre-minified and gzipped
  • 🔒 Reliable - 99.9% uptime SLA

📦 Package Manager

For projects with build systems, install via npm or yarn for better integration:

Terminal

Terminal

export default defineConfig({

      output: {

    }
  }
});

Quick Start Guide

Learn the fundamentals of using Lume UI components effectively in your projects.

🎯 Basic Concepts

Lume UI components are built as Web Components, which means they work like regular HTML elements but with enhanced functionality:

🏷️ HTML-like Syntax

Use components just like regular HTML tags with attributes for customization.

🔒 Shadow DOM

Styles are encapsulated and won't interfere with your existing CSS.

📡 Event System

Components emit custom events that you can listen to for interactions.

🎨 Theming System

Built-in dark/light themes with support for custom color schemes.

🌐 Global Theme Setting

You can set a global theme for your entire application:

JavaScript 
// Set global theme via CSS custom property
document.documentElement.style.setProperty('--lume-theme', 'dark');

// Or add a class to body
document.body.className = 'lume-theme-dark';

// Components will automatically inherit the global theme
// unless overridden with the theme attribute

📡 Event Handling

Lume UI components emit custom events that you can listen to for user interactions:

Click me for event demo
JavaScript 
// Listen to component events
const button = document.querySelector('lume-button');

// Lume UI custom event
button.addEventListener('lume-click', (event) => {
  console.log('Button clicked!', event.detail);
  // event.detail contains component information
});

// Standard events also work
button.addEventListener('click', (event) => {
  console.log('Standard click event');
});

// Form events for form components
const form = document.querySelector('lume-form');
form.addEventListener('lume-submit', (event) => {
  console.log('Form submitted:', event.detail.data);
  // Prevent default form submission if needed
  event.preventDefault();
});

📱 Responsive Design

Lume UI components are responsive by default, but you can customize breakpoints:

CSS 
/* Responsive customization */
@media (max-width: 768px) {
  lume-button {
    --lume-button-size: small;
    --lume-button-padding: 0.5rem 1rem;
  }
  
  lume-navbar {
    --lume-navbar-mobile-breakpoint: 768px;
  }
}

@media (max-width: 480px) {
  lume-hero {
    --lume-hero-heading-size: 2rem;
    --lume-hero-padding: 1rem;
  }
}
