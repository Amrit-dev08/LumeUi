LumeUI Documentation

A modern, lightweight web component library for building beautiful and interactive user interfaces.

Welcome to LumeUI

LumeUI provides a collection of customizable, framework-agnostic components that work seamlessly across all modern browsers. Built with vanilla JavaScript and Web Components, it offers modern design patterns with extensive theming capabilities.

Installation

Get started with LumeUI by including the CDN link in your HTML. No build process required! 

<script
  src="https://cdn.jsdelivr.net/gh/Amrit-dev08/LumeUi/dist/lume.js"
  type="module"
></script>

That's it! All components are now available as custom HTML elements.

Quick Start

Here's a complete example to get you started with LumeUI components. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>LumeUI Example</title>
    <script src="https://cdn.jsdelivr.net/gh/Amrit-dev08/LumeUi/dist/lume.js"></script>
  </head>
  <body>
    <lume-button variant="primary" size="large">
      Get Started
    </lume-button>

    <lume-hero
      heading-text="Welcome to LumeUI"
      description-text="Build beautiful web interfaces with ease"
    >
    </lume-hero>
  </body>
</html>

LumeButton

A versatile button component with multiple variants, sizes, and extensive customization options.

Default Usage

Click Me 
<lume-button>Click Me</lume-button>

Button Variants

The button component supports 7 different variants, each with unique styling.

Primary
Default brand-colored button for main call-to-action elements
Primary Button 
<lume-button variant="primary">Primary Button</lume-button>
Secondary
Muted gray styling for secondary actions
Secondary Button 
<lume-button variant="secondary">Secondary Button</lume-button>
Outlined
Transparent background with colored border
Outlined Button 
<lume-button variant="outlined">Outlined Button</lume-button>
Ghost
Minimal styling with subtle hover effects
Ghost Button 
<lume-button variant="ghost">Ghost Button</lume-button>
Glow
Features glowing shadow effect with customizable color
Glow Button 
<lume-button variant="glow" glow-color="rgba(37, 99, 235, 0.5)">Glow Button</lume-button>
Icon Button
Circular button design perfect for icons 
<lume-button variant="icon-btn" size="medium">
  <svg width="20" height="20" viewBox="0 0 20 20" fill="currentColor">
    <path d="M10 12l-4-4h8l-4 4z" />
                                  </svg>
</lume-button>
Link
Styled as underlined text, functions as clickable link
Link Button 
<lume-button variant="link" href="https://example.com">Link Button</lume-button>

Event System

All LumeUI components emit custom events for user interactions and state changes.

Button Events

// Button click events
document.addEventListener('lume-button-click', (event) => {
  console.log('Button clicked:', event.detail);
  // event.detail: { text, variant, size }
});

Form Events

// Form submission
document.addEventListener('form-submit', (event) => {
  console.log('Form submitted:', event.detail);
  // event.detail: { data, formData, type }
  
  // Prevent default submission if needed
  event.preventDefault();
});

Hero Events

// Hero button clicks
document.addEventListener('lume-hero-primary-click', (event) => {
  console.log('Hero primary button clicked:', event.detail);
});

document.addEventListener('lume-hero-secondary-click', (event) => {
  console.log('Hero secondary button clicked:', event.detail);
});
